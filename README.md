# Student Management System in C++

A comprehensive console-based student management application developed in C++ as a first-semester academic project.

## Project Overview

This system manages student registration and academic records for an educational institution with role-based administrative control.

## Features

### Student Registration
- Personal information management
- Academic record registration
- Program selection:
  - Artificial Intelligence
  - Computer Science
  - Software Engineering
  - Data Science

### Academic Levels
- Undergraduate
- Graduate
- Postgraduate

### Performance Tracking
- Intermediate marks
- GPA / CGPA evaluation
- Automatic scholarship calculation

### Administrative Controls
- View all records
- Search students by name
- Update student data
- Remove student records

### Facilities Management
- Hostel allocation
- Transport management

### Security & Validation
- Input validation
- Admin authentication
- Data verification

## Technologies Used

- C++
- Arrays
- Functions
- Loops & Conditionals
- String Handling
- iomanip
- Structured Programming
